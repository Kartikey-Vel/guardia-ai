# Guardia AI Backend Scaffold

This backend directory is scaffolded for a FastAPI + AI pipeline project.

Important:
- This scaffold intentionally contains templates only.
- No production or business logic has been implemented yet.
- Use this as the starting structure for team development.

## Folder Layout
- api/: Route module templates
- models/: Data model and schema templates
- ai/: AI integration templates
- websocket/: Real-time communication templates
- utils/: Shared utility templates

## Quick Setup
Follow the root setup guide:
- ../README_BACKEND_SETUP.md

## Environment Files
- .env.example: Environment variable template
- requirements.txt: Runtime dependencies
- requirements-dev.txt: Development dependencies
